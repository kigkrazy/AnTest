# 安卓项目初始化框架
本框架是为了方便开发，整合一些最基本的功能。在开发新项目的时候可以基于这个项目进行修改。

## 项目介绍
项目引用了几个大框架`Design + MVP + RxJava2 + Retrofit + Dagger2 + QMUI`进行项目初始化搭建。

## 使用方法
直接下载项目，修改项目`包名`，`项目名`，`applicationId`等关键信息就可以进行开发了。
